# 5th Semester Web Technology Assignments

This repository contains all the assignments for the 5th Semester Web Technology course.

## Assignments

### Assignment 1: Personal Portfolio Website

A responsive website featuring a professional resume and personal biodata.

- [View Assignment 1](Assignment%201/index.html)
- [Assignment 1 README](Assignment%201/README.md)

### Assignment 2: Online Registration Form

A web application built with HTML, CSS, JavaScript, jQuery, and PHP for an online registration form. The form accepts user information and displays it with formatting/styling upon successful submission.

- [View Assignment 2](Assignment%202/index.html)
- [Assignment 2 README](Assignment%202/README.md)

## Technologies Used

- HTML5
- CSS3
- JavaScript
- jQuery
- PHP

## How to View

To view any assignment, simply navigate to the assignment folder and open the `index.html` file in your web browser, or run a local server:

```bash
python -m http.server 8000
```

Then visit `http://localhost:8000` in your browser and navigate to the desired assignment folder.

For Assignment 2, you'll need a server with PHP support to process the form correctly.
