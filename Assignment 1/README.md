# Web Technology Assignment 1

This project is a responsive personal portfolio website created as part of the 5th Semester Web Technology Assignment 1.

## Project Overview

The website contains multiple web pages featuring:
- A professional resume showcasing skills and experience
- Detailed personal biodata
- Responsive design that works on all device sizes
- Interactive elements powered by CSS and JavaScript

## File Structure

```
Assignment 1/
├── index.html          # Main homepage
├── resume.html         # Professional resume page
├── biodata.html        # Personal biodata page
├── styles.css          # Styling for all pages
├── script.js           # JavaScript functionality
└── README.md           # This file
```

## Features Implemented

### HTML Structure
- Semantic HTML5 elements for better accessibility and SEO
- Properly structured content with headings, sections, and articles
- Navigation between all pages

### CSS Styling
- Responsive design using Flexbox and Grid layouts
- Modern UI with gradient backgrounds and card-based components
- Mobile-first approach with media queries for different screen sizes
- Consistent color scheme and typography

### JavaScript Functionality
- Mobile navigation toggle for small screens
- Smooth scrolling for anchor links
- Interactive hover effects and animations
- Dark/light mode toggle
- Form validation (for future implementation)

## Pages

### 1. Homepage (index.html)
- Welcome section with call-to-action buttons
- Brief description of the website content
- Navigation to other pages

### 2. Resume Page (resume.html)
- Professional profile header with contact information
- Skills section organized by technology categories
- Work experience with detailed responsibilities
- Education history and certifications

### 3. Biodata Page (biodata.html)
- Personal information table
- Contact details
- Educational background
- Family information
- Languages known
- Hobbies and interests
- Personal statement

## Technologies Used

- **HTML5**: For semantic markup and structure
- **CSS3**: For styling, animations, and responsive design
- **JavaScript**: For interactive elements and DOM manipulation
- **jQuery**: For simplified JavaScript operations

## Responsive Design

The website is fully responsive and adapts to different screen sizes:
- Desktop: Multi-column layout with full navigation
- Tablet: Adjusted column spans and reduced margins
- Mobile: Single column layout with hamburger menu

## How to View the Website

1. Clone or download this repository
2. Navigate to the `Assignment 1` folder
3. Open `index.html` in your web browser
4. Alternatively, run a local server:
   ```bash
   python -m http.server 8000
   ```
   Then visit `http://localhost:8000` in your browser
